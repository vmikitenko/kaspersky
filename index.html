Информационная безопасность АРМ

Для обеспечения информационной безопасности необходимо

  • Установить Антвирус Касперского версии 11.1.1.126
  • Установить агент администрирования версии 10.5.1781
  • Настроить соединение агента администрирования с сервером KSC 172.26.27.4 (VOL1-KMS)
  • Установить ключ активации Антивируса Касперского
  • Запустить однократно обновление антивирусных баз
  • Запустить однократно полную проверку на вирусы
  • Проверить соединение с сервером устновки обновлений Windows "WSUS"
  • Удаленно установить СЗИ Secret Net Studio
